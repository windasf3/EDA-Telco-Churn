### Telco Customer Churn Analysis

This project focuses on Exploratory Data Analysis (EDA) of Telco customer churn dataset. The code provided in this repository includes data preprocessing steps, handling missing values, and conducting EDA and univariate analysis to gain insights into the percentage of customers who churn.


#### Results

The results of the EDA and univariate analysis will be presented in the `results` directory. This includes visualizations such as histograms, bar plots, and summary statistics to understand the factors influencing customer churn.
